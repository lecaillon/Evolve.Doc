#### Evolve .NET library
- .NET 4.7.2+
- .NET Standard 2.0 / .NET 5.0

#### Evolve .NET tool
- .NET Core SDK 3.0 and later

#### Evolve CLI
- Windows-64bit
- Linux-64bit
